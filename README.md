<h1 align="center">CoreGetGame</h1>

鉱石を取得することでポイントを取得できるゲームを実現するプラグイン。

<table>
    <tr>
        <th>コマンド</th><th>説明</th>
    </tr>
    <tr>
        <td>/game start</td><td>ゲーム開始コマンド</td>
    </tr>
    <tr>
        <td>/game end</td><td>ゲーム終了コマンド</td>
    </tr>
</table>

# 動作環境
- Spigot 1.18.2
- Oracle OpenJDK version 17.0.4

# 仕様
- [ ] 鉱石ごとにポイントを設定可能
- [ ] クリアする条件(ポイント)を設定可能
- [ ] クリア時にコマンドを実行可能
- [ ] コマンドから設定変更可能
